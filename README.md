## Features

- os, root, path, git, executiontime, status, rust, node, go, python, time, text, prompt.

---

## Requirements

Before using this theme, make sure you have:

* [Oh My Posh](https://ohmypo.sh/) installed
* [A Nerd Font](https://www.nerdfonts.com/font-downloads) installed and configured in your terminal (or fonts that that support for Nerdfont's icons)

---

## Installation

1. Clone the repository:
```bash
git clone https://github.com/valkyrieglasc/marathon-ohmyposh-config.git
```
> Copy the theme to your preferred Oh My Posh themes directory and update your shell configuration accordingly.

<br>

2. Edit your current OhMyPosh config name in your terminal
```bash
nvim ~/.zshrc
```
> Run `echo $0` in your terminal to see which shell you're using

<br>

3. Find and edit the line below
```
eval "$(oh-my-posh init zsh --config '~/your-directory/marathon-omp.json')"
```
<br>

4. Reload the shell and you're done!
